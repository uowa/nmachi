1. にゃしぃフレンズとは
にゃしぃフレンズは、
フリーの丸文字っぽい手書きフォントです。
ブラウザゲーム『艦隊これくしょん -艦これ-』に
登場するアイテム「『二周年記念』掛け軸」っぽさを
そこはかとなく醸し出せるよう、
一字一字、心を込めて仕上げた……のですが、
某睦月型駆逐艦2番艦に細部をif改装されたという設定で、
様々な文字・記号がハートマークまみれになってしまいました。

結果的に、掛け軸を書いたとされる原作キャラクター「睦月」の
元気で優しく、明るい雰囲気に加え、
そこはかとなく色気や意味深な雰囲気を漂わせる代物に仕上がりました。
本家にゃしぃフォントともども、ご愛用いただければうれしいです。



2. 収録文字
・ひらがな
・カタカナ…全角/半角
・アルファベット…全角/半角
・数字…全角/半角
・記号…ギリシア文字、キリル文字、その他一式網羅。アクサンにも対応
・漢字…JIS第一/第二水準6,355字＋IBM拡張文字
	＋日本漢字能力検定1級配当漢字（一部を除く）＋α　計：約7,000字



3. 動作環境
現在のところ、
Microsoft Windows XP、Vista、7、8、10、11、
およびMac OS Xで動作を確認しております。

web拍手やXなどでご一報いただければありがたいです。

なお、
